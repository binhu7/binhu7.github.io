This section of ECE 329 will follow the Mathematica based approach introduced in Fall 2015 by Prof. Dan Wasserman. You will find in the Course Calendar a link to a Mathematica notebook for each class. These notebooks will be partially evaluated during class and will be completed after class by you and submitted via Compass2g before the due time. Questions regarding Mathematica can be done via Piazza. (to sign up for this class in Piazza: Piazza sign-up). No prior experience with Mathematica is required --- the focus of the class will be on using basic Mathematica techniques for understanding/illustrating E&M, as opposed to using E&M to teach Mathematica. If you are new to Mathematica watch the following videos prior to starting your first Mathematica assignment: Notebooks Methods to Get Started Basic Calculations Basic Graphics Making Interactive Models

All engineering students have free access to Mathematica via Citrix remote server. In preparation to class download and install the Citrix Receiver software to your mobile device (laptop or tablet) from Citrix Download or Citrix Download for Linux and bring your mobile device to class everyday.
